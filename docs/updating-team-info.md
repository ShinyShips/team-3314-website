# Updating the Team Info page — no coding required

The **Team Info** page on [frc3314.com/team-info/](https://frc3314.com/team-info/)
shows two things that update themselves:

1. **The calendar** — comes straight from the team's Google Calendar.
2. **The documents** — come straight from the team's Google Drive folder.

You never touch the website. You just use Google Calendar and Google Drive,
and the page updates within moments.

## ⚠️ The one rule

**Everything on this page is public on the internet.** Anyone in the world
can open these documents and see these events. Never put anything in the
calendar or the Drive folder that contains personal information: no rosters
with phone numbers or emails, no medical forms, no home addresses, no
permission slips that have been filled in. Blank forms are fine; filled-in
ones are not.

## Adding or changing a date

1. Open [Google Calendar](https://calendar.google.com) signed in to your
   team-account access.
2. Make sure you're adding the event to the **team calendar** (pick it in
   the "Calendar" dropdown when creating the event — not your personal one).
3. Add the event like any other: title, date, time, location, and a
   description if you want. All of those appear on the website.

That's it. Edits and deletions work the same way.

## Adding a document

1. Open the **Team Website Documents** folder in
   [Google Drive](https://drive.google.com).
2. Drag the file in (or click **New → File upload**).

The file appears on the website automatically, listed by name.

- **To put it under a heading** (like "Forms" or "Parent Info"), drop it
  into that subfolder instead of the main folder.
- **To make a new heading**, create a new subfolder — the website turns
  every subfolder into a section with that name.
- **To remove a document from the site**, just delete it (or move it out of
  the folder).
- **Renaming** a file or folder renames it on the site too.

## If something looks wrong

If the page ever shows "couldn't load," it usually fixes itself in a few
minutes. If a document shows "You need access" when opened, its sharing got
changed — right-click the file in Drive → **Share** → make sure it's
"Anyone with the link — Viewer" (it inherits this from the folder
automatically unless someone changed it).
